## 正则表达式

正则表达式很基本，很多编程语言都内嵌正则表达式的语法，他们的语法格式很像，只是标示的符合不同而已。这里我们主要学习linux中使用的格式。

* 字符匹配
  ```
  [abc]             匹配a或者b或者c
  .                    匹配任意字符
  [:lower:]       小写字母，等于[a-z]
  [:upper:]       大写字母，等于[A-Z]
  [:alpha:]       全部字母
  [:alnum:]      字母和数字
  [:punct:]       全部符号
  ```
* 次数匹配
  ```
  \+                 一次以上
  \?                  0次或1次
  *                  任意次
  ```
* 位置锚定
  ```
  ^                  行首锚定
  $                   行尾锚定
  \<                 词首锚定
  \>                 词尾锚定
  ```
* 分组
  ```
  \(\)                把某部分内容作为一个组来锚定
  |                    把组内的内容分为多个块，多个块匹配到其中一个就显示
  ```
